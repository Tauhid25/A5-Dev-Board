Question-1: What are the different ways to select an element in the DOM?

There are different ways to select an element in the DOM.Here are: (a) Using getElementById : This method allows to select an element by its ID.It is most useful method.(b) Using getElementsByTagName : This method allows to select elements by their class name. It returns a collection of all elements in the document.(c) Using getElementsByTagName : This method allows to select elements by their tag name. It returns a collection of all elements in the document.(d) Using querySelector : The querySelector method allows you to select the first element that matches a specified CSS selector(s) in the document.(e) Using querySelectorAll : The querySelectorAll method allows to select all elements that match a specified CSS selector(s) in the document.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The differences between innerHTML, innerText, and textContent are :(a) innerHTML:It gets or sets the full HTML code.It parses the content as HTML, so it can include and render HTML tags.(b)innerText:Itgets or sets the inner text of an element.(c)textContent:It gets or sets the text content of an element, including all elements, with hidden element.It ignores HTML markup and CSS styling.It is generally faster than innerText.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a concept in the DOM (Document Object Model). It happens when an element receives an event, and that event bubbles up (or you can say is transmitted or propagated) to its parent and ancestor elements in the DOM tree until it gets to the root element.Event bubbling is the propagation of events from the bottom to the top of the DOM tree, from child elements to parent elements. It is one sort of event propagation. Another form is event capturing, which occurs when events propagate down the DOM tree from parent to child elements.
